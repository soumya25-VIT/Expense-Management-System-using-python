# Expense-Management-System-using-python
Managing daily expenses for Budgeting and Financial Planning using python

Project Overview:
Managing daily expenses is important for Budgeting and Financial planning.
This program allows users to:
•	Add new expenses
•	View all saved expenses
•	Calculate the total expense

Objectives: 
•  Understand the concept of variables, lists, functions, loops, and conditionals.
•  Implement a menu-driven application using Python.
•  Apply basic problem-solving and modular programming concepts.
•  Learn how to structure a simple console-based application.

Features:
✔ Add an expense
✔ Store category, amount, and note
✔ View all expenses
✔ Calculate total expense
✔ Simple & beginner-friendly
✔ Modular code with functions
